## Development

This project focuses on using VS Code as an IDE for development on all platforms. No Android Studio, Xcode, or other IDE required.

```shell
cd src
flutter pub get
```

### Windows

```shell
cd src/windows
```

### macOS

```shell
brew install swiftlint
brew install swiftformat
```

```shell
cd src/macos
```

### Linux

```shell
cd src/linux
```

### Android

```shell
cd src/android
gradle wrapper
gradle build
```

### iOS

```shell
cd src/ios
```
